Your Energy — це сучасний односторінковий вебзастосунок (SPA), розроблений як курсова робота з дисципліни «Основи веброзробки (JavaScript)». Проєкт реалізовано на основі професійного Figma-макета з використанням живого бекенду через Swagger REST API. Головна мета застосунку — надати користувачам зручний інструмент для пошуку фізичних вправ, перегляду детальних інструкцій та формування власного списку тренувань. Завдяки мобільно-орієнтованому підходу (Mobile-first) та семантичній верстці, сервіс забезпечує високу доступність та коректне відображення на будь-яких пристроях, підтримуючи керування як через тачскрін, так і за допомогою клавіатури.

Центральним елементом функціональності є розгалужена система пошуку та фільтрації. Користувачі можуть сортувати вправи за цільовими групами м'язів, частинами тіла або необхідним обладнанням. Для зручності передбачено пошук за ключовими словами всередині обраних категорій, а також систему швидких переходів через хештеги на головному екрані. Весь каталог працює на базі серверної пагінації, що гарантує швидке завантаження даних, а візуальний контент оптимізовано за допомогою «лінивого» завантаження (lazy loading) та адаптивних зображень.

Взаємодія з конкретною вправою відбувається через продуману систему модальних вікон. У детальному перегляді користувач бачить повну характеристику тренування: від спалених калорій та часу виконання до рейтингу популярності та текстового опису. Тут же реалізована можливість додавання вправи до списку «Обраного», який зберігається у локальному сховищі браузера (localStorage), що дозволяє зберігати дані навіть після перезавантаження сторінки. Окремий розділ «Favorites» динамічно оновлюється, пропонуючи користувачеві зручний інтерфейс для управління своїм планом занять.

Соціальна та інтерактивна складова проєкту включає систему оцінювання та підписки. Користувачі можуть виставляти рейтинг вправам через спеціальну форму з валідацією даних, яка відправляє PATCH-запити на сервер. Система сповіщень (Global Notification) миттєво інформує про успіх операції або помилки, наприклад, при спробі повторної підписки на розсилку через футер сайту. Додатковим елементом залучення є віджет «Цитата дня», який отримує дані з API та кешує їх на 24 години, забезпечуючи щоденну порцію мотивації без зайвих запитів до мережі.

З технічної точки зору проєкт побудований на сучасному стеку технологій без використання важких фреймворків, що підкреслює майстерність володіння Vanilla JavaScript. Структура коду є модульною: стилі організовані за допомогою SCSS із чітким розподілом на компоненти, а збірка проєкту здійснюється інструментом Vite. Використання делегування подій (event delegation) та семантичних тегів HTML5 робить застосунок швидким, надійним та зручним для розширення в майбутньому.
